Things don't work on their own

A website is made up of lots of parts. All computer software is!

  1. A user interface - the bit anyone visiting your website sees and can use (HTML, CSS and JavaScript)
  2. The code that runs the website (Python, Java, Ruby, PHP)
    • You don't need this, a website can just be HTML, CSS and maybe some JavaScript. That's called a 'static' website
  3. A database to store information in
  4. The computer (server) the website runs on
    (We won't talk about that for now)
